Portable • Windows Native • Local-First

Your AI agent,

SMF Forge Desktop is a portable AI agent that lives on your machine — no cloud dependency, no WSL required. It learns your projects, builds its own skills, and works with the tools you already use.

> pip install smf-forge-desktop

Requires Python 3.11+ on Windows 10/11. Installs everything automatically — no WSL, no Docker.

Download on GitHub

Get started in 60 seconds

1

Install

pip install smf-forge-desktop

Installs the agent engine, Tkinter GUI, and bundled skills. No external Python or WSL needed.

2

Configure

cmd / PowerShell 
# Interactive setup wizard
smf-forge setup

# Or choose your model
smf-forge model

Connect OpenRouter, Kimi, or any OpenAI-compatible API. Local models via LM Studio supported out of the box.

3

Launch the GUI

cmd 
smf-forge-gui

Opens the native Windows Tkinter interface — or run smf-forge for the CLI. Both share the same memory and session state.

4

Go multi-platform (optional)

cmd 
# Interactive gateway setup wizard
smf-forge gateway setup

# Start the messaging gateway
smf-forge gateway

# Install as a Windows service
smf-forge gateway install

Walk through connecting Telegram, Discord, Slack, or WhatsApp. Runs as a Windows service in the background.

5

Keep it up to date

cmd 
smf-forge update

Pulls the latest changes and reinstalls dependencies. Run anytime to get new features and fixes.

Native Windows support is first-class. No WSL required.

Features

Lives Where You Do

Telegram, Discord, Slack, WhatsApp, and CLI from a single gateway — start on one, pick up on another.

Grows the Longer It Runs

Persistent memory and auto-generated skills — it learns your projects and never forgets how it solved a problem.

Scheduled Automations

Natural language cron scheduling for reports, backups, and briefings — running unattended through the gateway.

Native Windows GUI

A polished Tkinter desktop app with extension manager, permission controls, and DPI scaling for modern displays.

Runs Local

No cloud lock-in. Your agent runs on your hardware with your API keys. LM Studio, OpenRouter, Kimi, or self-hosted endpoints.

100+ Tools Out of the Box

Terminal, file system, browser automation, vision, image generation, code execution, subagent delegation, memory, and skill creation.

Tools

40+ built-in — web search, terminal, file system, browser automation, vision, image generation, text-to-speech, code execution, subagent delegation, memory, task planning, cron scheduling, multi-model reasoning, and more.

Platforms

Telegram, Discord, Slack, WhatsApp, Signal, Email, and CLI — all from a single gateway. Connect to OpenRouter, Kimi, or any OpenAI-compatible API.

Environments

Run locally, in Docker, over SSH, on Modal, Daytona, or Singularity. Container hardening with read-only root, dropped capabilities, and namespace isolation.

Skills

40+ bundled skills covering MLOps, GitHub workflows, research, and more. The agent creates new skills on the fly and shares them via the open agentskills.io format. Install community skills from ClawHub, LobeHub, and GitHub.

Research

Batch trajectory generation with parallel workers and checkpointing. Atropos integration for RL training. Export to ShareGPT for fine-tuning with trajectory compression.
